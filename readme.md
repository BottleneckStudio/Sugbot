# Sugbot

Is a bot for sharing the most recent local news from local news agencies using RSS feeds.

### Technology Stack:
  - Node.js
  - ExpressJS


##### Testing Tools:
  - Chai
  - Mocha

### Installation

Clone the repo

```sh
$ cd Sugbot
$ npm install
```

To boot-up the server:

```sh
$ npm start
```

To run tests:
```sh
$ npm test
```
